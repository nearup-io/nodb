# nodb

Open source API for app development with database, auth and vector search.

Prerequisites:

1. [Install Bun](https://bun.sh/docs/installation)
2. Copy `.env.example` to your `.env` file and set your variables

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```
